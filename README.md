# Project-Pemrograman-Web


# Cara Menjalankan Project

1. Extract file RAR/ZIP.

2. Masuk ke folder backend.

3. Jalankan server dengan perintah:
   frontend : npm run dev
   backend  : npm start

4. Setelah server berjalan, backend dapat diuji menggunakan Thunder Client atau Postman.
   Contoh endpoint yang bisa dicoba:
   - POST /api/user/register
   - POST /api/user/login
   - GET /api/items

Project ini baru proses di backend (frontend belum selesai).
